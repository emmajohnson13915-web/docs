JSONViewer AI helps users view, format, and understand JSON easily.

This documentation explains how to get started, use core features, and get the most out of the product.

## What is JSONViewer AI?

JSONViewer AI is a simple tool that helps you:
- Format raw JSON
- Read large JSON files clearly
- Copy and share structured output

It is designed for both developers and non-technical users who work with JSON data.

## Who is this documentation for?

This documentation is useful for:
- Developers working with APIs or JSON files
- Product managers reviewing structured data
- Anyone who needs to quickly understand JSON

## Getting started

To get started:
1. Open JSONViewer AI
2. Paste your JSON data
3. View and format the output instantly

No setup or configuration is required.
